## 2. 核心玩法

### 2.1 游戏类型
- **横版自动推进** + **即时战斗**
- 主角自动向右移动，玩家操控战斗

### 2.2 核心循环
1. 主角自动向右前进
2. 刷出小怪阻挡
3. 主角击杀小怪获得经验
4. 经验满升级，提升属性
5. 继续前进，循环

## 3. 角色系统

### 3.1 主角

| 属性 | 初始值 | 说明 |
|-----|-------|-----|
| 等级 | 1 | 初始等级 |
| 生命值 | 100 | HP |
| 攻击力 | 10 | 造成伤害 |
| 移动速度 | 150 | 像素/秒 |
| 攻击速度 | 1.0 | 次/秒 |
| 经验值 | 0 | 当前经验 |
| 升级经验 | 100 | 升级所需 |

#### 武器
- **初始武器：剑**
  - 攻击范围：80像素
  - 攻击伤害：10
  - 攻击间隔：1秒

### 3.2 怪物

| 怪物类型 | 生命值 | 攻击力 | 经验值 | 出现场景 |

## 4. 战斗系统

### 4.1 攻击方式
- 主角自动攻击范围内的敌人
- 剑武器为近战攻击

### 4.2 伤害计算
```
伤害 = 攻击力
```

### 4.3 死亡判定
- 主角HP≤0：游戏结束
- 怪物HP≤0：怪物死亡，掉落经验

---

## 5. 升级系统

### 5.1 升级规则
- 经验累计达到升级阈值时自动升级
- 升级后：等级+1，生命值回满，攻击力+2

### 5.2 升级公式
```
升级所需经验 = 100 × 等级
```
