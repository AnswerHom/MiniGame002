# MiniGame002 v1.5.6 设计文档

## 版本主题
背包UI系统 - 增加背包界面入口与基础功能

---

## 1. 需求来源

**优化方向列表第1条**：增加背包UI入口 - 点击弹出背包界面

---

## 2. 功能需求

### 2.1 背包入口

- 在游戏主界面添加背包按钮（建议位置：右上角或底部菜单栏）
- 按钮样式：图标+文字，尺寸≥44px点击区域
- 点击后弹出背包界面面板

### 2.2 背包界面

- **界面尺寸**：覆盖屏幕60%-80%，居中显示
- **布局**：
  - 顶部标题栏：显示"背包"标题，关闭按钮（X）
  - 物品网格区：4x4或6x4格子，每格可放一件物品
  - 底部状态栏：显示负重/容量（如：5/20）
- **背景**：半透明遮罩，点击遮罩或关闭按钮退出
- **动画**：弹出/收起带缓动效果（200-300ms）

### 2.3 物品显示

- 物品图标：64x64像素
- 物品类型：武器、防具、消耗品、素材
- 数量显示：堆叠物品显示数量角标
- 物品选中态：点击高亮显示

### 2.4 交互功能

- 点击物品：显示物品详情（名称、描述、属性）
- 拖拽物品：可拖动整理背包位置
- 物品使用：消耗品点击使用，装备点击穿戴

## 3. 技术实现

### 3.1 文件结构

```
js/
  ├── ui/
  │   ├── backpack.js      # 背包UI逻辑
  │   ├── backpack.css     # 背包样式
  │   └── item.js          # 物品数据与渲染
  └── game.js              # 集成背包入口
```

### 3.2 数据结构

```javascript
// 物品数据结构
{
  id: "item_001",
  name: "铁剑",
  type: "weapon",
  icon: "sword.png",
  count: 1,
  attrs: { atk: 5 },
  desc: "一把普通的铁剑"
}

// 背包数据结构
{
  capacity: 16,
  items: []
}
```

## 4. UI设计规范

- **色彩**：深色半透明背景（rgba(0,0,0,0.8)），边框高亮
- **字体**：16px-20px，浅色文字
- **按钮**：最小44px点击区域，hover/active反馈
- **动画**：ease-out缓动，弹出scale 0.9→1
