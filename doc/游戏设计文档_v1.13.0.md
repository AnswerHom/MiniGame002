## 2. 迭代改进建议（v1.13.0 新需求）

### 2.1 UI模块拆分（P1优先级）

#### 2.1.1 ui.js - UI界面模块
将所有UI绘制逻辑从 game.js 提取：
- 状态栏（生命值、经验条、境界显示）
- 背包界面（装备格子、物品列表）
- 技能栏（技能图标、冷却显示）
- 副本选择界面
- 升级/突破弹窗

#### 2.1.2 systems.js - 系统配置分离
将装备数据库、武器数据等提取：
```javascript
// WEAPONS, ARMORS, CONSUMABLES 等
```

### 2.2 性能优化

#### 2.2.1 渲染优化
- 粒子系统对象池
- 视锥剔除（只渲染屏幕内对象）
- 分层渲染优化

#### 2.2.2 游戏循环优化
- requestAnimationFrame 精细化控制
- 减少每帧 GC 操作

### 2.3 存档系统增强

#### 2.3.1 自动存档
- 每30秒自动保存
- 退出时强制存档

#### 2.3.2 多存档槽位
- 支持3个存档槽位

### 2.4 新手引导

#### 2.4.1 交互式引导
- 首次进入游戏弹出引导面板
- 逐步展示核心玩法
