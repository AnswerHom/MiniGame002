# MiniGame002 v2.7.0 设计文档

## 版本主题
增加主角属性面板与主界面入口

---

## 1. 需求来源

根据未实现需求列表第2条：增加主角属性面板和主界面入口

---

## 2. 问题分析

### 2.1 当前问题
- 玩家无法查看详细属性
- 缺乏属性面板入口
- 不方便了解角色成长情况

### 2.2 期望效果
- 主界面增加属性面板入口按钮
- 点击可查看主角详细属性
- 属性实时更新显示

---

## 3. 功能设计

### 3.1 入口按钮
- **位置**：主界面右上角系统栏
- **样式**：图标+文字 "属性"
- **尺寸**：44×44px（符合触屏标准）
- **反馈**：点击有按压效果

### 3.2 属性面板
- **打开方式**：点击入口按钮弹出
- **关闭方式**：点击遮罩/关闭按钮/按ESC
- **位置**：屏幕居中
- **尺寸**：宽度300px，高度自适应
- **背景**：半透明黑色遮罩

### 3.3 面板内容
| 属性 | 说明 |
|------|------|
| 等级 | 主角当前等级 |
| 经验 | 当前经验/升级所需经验 |
| 生命 | HP上限/当前HP |
| 攻击力 | 基础攻击+装备加成 |
| 防御力 | 基础防御+装备加成 |
| 暴击率 | 百分比显示 |
| 移动速度 | 数值显示 |
| 武器类型 | 当前装备武器 |

### 3.4 动画效果
- 弹出：0.2秒缩放+淡入
- 关闭：0.15秒缩放+淡出
