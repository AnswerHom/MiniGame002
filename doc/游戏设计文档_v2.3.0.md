# 游戏设计文档 v2.3.0

## 版本信息
- **版本号**: v2.3.0
- **需求类型**: 灵兽系统 - 灵兽捕捉与孵化
- **创建时间**: 2026-03-06
- **优化方向**: 灵兽/伙伴系统（v2.2.0迭代）

---

## 需求概述

本版本是v2.2.0灵兽系统的第一阶段实现，专注于灵兽捕捉与孵化功能。玩家击败怪物后有机会获得灵兽蛋，孵化后获得灵兽作为战斗伙伴。

---

## 功能清单

### 1. 灵兽蛋掉落
- 击败怪物后一定概率掉落灵兽蛋
- 不同境界怪物掉落概率和品质不同
- 灵兽蛋掉落时有飘字提示和特效

### 2. 灵兽蛋数据结构
- 灵兽蛋ID
- 品质（普通/优秀/稀有/史诗/传说）
- 所属境界
- 孵化所需时间（秒）

### 3. 灵兽蛋仓库
- UI显示灵兽蛋仓库界面
- 展示已获得的灵兽蛋列表
- 显示灵兽蛋品质和状态
- 一键孵化功能

### 4. 灵兽孵化
- 点击灵兽蛋进行孵化
- 孵化动画和倒计时
- 孵化成功后获得随机灵兽
- 灵兽自动进入灵兽栏

### 5. 灵兽栏
- 玩家可查看已拥有的灵兽
- 选择一只灵兽作为战斗伙伴
- 显示灵兽属性和状态

## 数值设计

### 灵兽蛋掉落概率
| 怪物境界 | 掉落概率 | 品质分布 |
|---------|---------|---------|
| 炼气期 | 5% | 普通80%、优秀19%、稀有1% |
| 筑基期 | 8% | 普通60%、优秀35%、稀有5% |
| 金丹期 | 12% | 普通30%、优秀50%、稀有19%、史诗1% |
| 元婴期 | 15% | 普通10%、优秀40%、稀有40%、史诗10% |
| 化神期 | 20% | 普通5%、优秀30%、稀有45%、史诗19%、传说1% |
| 渡劫期 | 25% | 普通0%、优秀20%、稀有45%、史诗30%、传说5% |

### 孵化时间
| 品质 | 孵化时间 |
|------|---------|
| 普通 | 3秒 |
| 优秀 | 5秒 |
| 稀有 | 8秒 |
| 史诗 | 15秒 |
| 传说 | 30秒 |

- `js/beast.js` - 新建灵兽模块（灵兽蛋、灵兽类）

- `js/ui/ui.js` - 添加灵兽蛋仓库界面
- `index.html` - 引入beast.js
