## 1. 怪物形象实现（v1.10.1需求落地）

### 1.1 阴魂（鬼魂类）- 绘制实现

#### 游魂
- 白色半透明圆形鬼火，飘忽不定
- 大小30px，有淡淡蓝光
- 绘制：白色半透明圆形 + 蓝色光晕效果

#### 恶灵
- 青灰色实体，长发遮面
- 双臂下垂滴血，颜色#4a5568
- 轮廓清晰
- 绘制：青灰色椭圆形身体 + 黑色长发 + 红色血滴

#### 厉鬼
- 红色眼睛，周身黑气缠绕
- 獠牙外露，颜色#1a202c
- 血色光晕
- 绘制：深色身体 + 红色发光眼睛 + 黑色气场 + 獠牙

### 1.2 妖狼（兽类）- 绘制实现

#### 苍狼
- 灰色毛皮，体型瘦长
- 尾巴蓬松，眼神绿光
- 40px高
- 绘制：灰色身体 + 蓬松尾巴 + 绿色眼睛

#### 猛虎
- 黄色毛皮（#d69e2e），黑色条纹
- 体型健壮，额头"王"字纹
- 绘制：黄色身体 + 黑色条纹 + 王字纹 + 威严眼神

#### 獠牙猪
- 黑色毛皮，巨型獠牙向上弯曲
- 皮肤粗糙，体型肥硕
- 绘制：黑色身体 + 巨型向上獠牙 + 粗糙皮肤纹理

### 1.3 毒蛛（虫类）- 绘制实现

#### 毒蛛
- 黑色背甲，红色腹部斑点
- 8条细长腿
- 毒性标志-紫色纹路
- 绘制：黑色椭圆形背甲 + 红色斑点腹部 + 8条细腿 + 紫色毒纹

#### 蜈蚣
- 红褐色，多节身体
- 每节一对足，共15对足
- 头部红色毒颚
- 绘制：红褐色分段身体 + 多对足 + 红色毒颚

#### 蝎子
- 深紫色，弯曲毒尾
- 螯肢粗大，体型扁平
- 尾部毒针发光
- 绘制：深紫色扁平身体 + 大螯肢 + 弯曲毒尾 + 发光毒针

### 1.4 僵尸（尸类）- 绘制实现

#### 僵尸
- 灰绿色皮肤（#68d391）
- 双臂平伸，服装褴褛
- 眼眶深黑
- 绘制：灰绿色身体 + 平伸双臂 + 褴褛衣服 + 深黑眼眶

#### 跳尸
- 深绿色，皮肤腐烂
- 双臂前伸，跳跃移动
- 带血迹
- 绘制：深绿色腐烂身体 + 前伸双臂 + 跳跃姿态 + 血迹

#### 铜甲尸
- 青铜色盔甲（#b7791f）
- 身体僵硬，盾牌武器
- 金属质感
- 绘制：青铜色盔甲身体 + 盾牌 + 金属反光

## 2. 怪物境界光晕实现

### 2.1 光晕颜色规则
| 境界 | 光晕颜色 | 实现方式 |
|------|----------|----------|
| 练气 | 无 | 不绘制光晕 |
| 筑基 | 淡绿光 | 绿色半透明外发光 |
| 金丹 | 蓝光 | 蓝色半透明外发光 |
| 元婴 | 青光 | 青色半透明外发光 |
| 化神 | 黄光 | 黄色半透明外发光 |
| 炼虚 | 橙光 | 橙色半透明外发光 |
| 合体 | 红光 | 红色半透明外发光 |
| 渡劫 | 紫光 | 紫色半透明外发光 |

## 3. 战斗表现优化

### 3.1 打击特效增强
- 命中时增加粒子爆发效果
- 暴击时播放更大范围特效
- 不同武器类型有不同的打击特效

- 死亡时化作光芒消散
- 精英怪死亡有特殊动画
- BOSS死亡有专属过场动画

### 3.3 受伤反馈优化
- 受击时怪物会有闪白效果
- 血条抖动效果
- 击退效果增强
