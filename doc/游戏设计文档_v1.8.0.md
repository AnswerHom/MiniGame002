# MiniGame002 v1.8.0 设计文档

## 版本主题
技能系统 - 玩家可学习和升级技能提升战力

---

## 1. 需求来源

**新优化方向列表第2条**：技能系统 - 玩家可学习和升级技能增强战斗能力

---

## 2. 功能需求

### 2.1 技能类型

- **主动技能**：需要手动触发
  - 剑斩：快速斩击，造成额外伤害
  - 护盾：短时间内提升防御力
  - 疾跑：短时间内提升移动速度
- **被动技能**：常驻生效
  - 战斗意志：提升暴击率
  - 体力增强：提升最大生命值
  - 修炼加速：获得更多经验值

### 2.2 技能学习

- 玩家达到特定等级解锁技能
- 技能可通过战斗或道具解锁
- 每个技能有独立的学习条件

### 2.3 技能升级

- 消耗游戏资源（经验/金币）升级技能
- 技能等级提升增强效果
- 技能有最高等级限制

### 2.4 技能界面

- 技能图标展示
- 技能等级显示
- 技能效果说明
- 升级按钮和资源消耗

---

## 3. 技术实现

### 3.1 数据结构

```javascript
// 技能数据结构
{
  id: "skill_001",
  name: "剑斩",
  type: "active", // active, passive
  icon: "skill_sword.png",
  level: 1,
  maxLevel: 5,
  unlockLevel: 1, // 解锁等级
  effect: { atkBonus: 10 }, // 效果参数
  upgradeCost: { exp: 100, gold: 50 }, // 升级消耗
  desc: "快速斩击，造成额外伤害"
}

// 玩家技能数据
{
  learned: ["skill_001"], // 已学习技能
  activeSkills: { skill_001: true }, // 当前激活的主动技能
  skillPoints: 0 // 技能点
}
```

### 3.2 文件结构

```
js/
  ├── skill.js         # 技能数据和逻辑
  └── ui/skillPanel.js # 技能界面
```

### 3.3 战斗集成

- 主动技能：点击释放，有冷却时间
- 被动技能：自动生效，无需操作
- 技能效果在战斗循环中实时计算
