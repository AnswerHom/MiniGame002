# MiniGame002 v3.8.0 设计文档

## 版本主题
左上角主角状态栏优化

---

## 1. 需求来源

根据未实现需求列表：左上角主角状态栏，只显示境界、等级、血条、经验条，布局不重叠

---

## 2. 问题分析

### 2.1 当前问题
- 状态栏信息可能重叠
- 布局不够整洁
- 信息展示不清晰

### 2.2 期望效果
- 垂直排列，层次分明
- 不重叠，整洁清晰
- 只显示必要信息

---

## 3. 界面设计

### 3.1 布局结构（垂直排列）
```
┌──────────────┐
│ 境界：练气   │  ← 第1行
├──────────────┤
│ Lv.5         │  ← 第2行
├──────────────┤
│ ████████░░  │  ← 第3行 血条
│ HP 80/100    │
├──────────────┤
│ ██████░░░░  │  ← 第4行 经验条
│ EXP 60/100   │
└──────────────┘
```

### 3.2 位置坐标
- X：20px（左侧安全边距）
- Y：20px（顶部安全边距）

### 3.3 尺寸规格
| 元素 | 尺寸 |
|------|------|
| 境界文字 | 14px |
| 等级文字 | 16px 加粗 |
| 血条 | 宽120px × 高12px |
| 经验条 | 宽120px × 高8px |
| 行间距 | 4px |

### 3.4 颜色方案
- 境界：白色
- 等级：黄色
- 血条：红色（满）→橙色（半）→红色（低）
- 经验条：蓝色
