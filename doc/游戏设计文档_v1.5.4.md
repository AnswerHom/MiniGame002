## 4. 技术实现

### 4.1 音频管理器

```javascript
// audio.js - 音频管理
class AudioManager {
    constructor() {
        this.enabled = true;
        this.volume = 0.7;
        this.sounds = {};
        this.music = null;
    }
    
    loadSound(name, src) {
        this.sounds[name] = new Audio(src);
    }
    
    playSound(name) {
        if (!this.enabled) return;
        this.sounds[name]?.play();
    }
    
    setVolume(value) {
        this.volume = value;
    }
    
    toggleMute() {
        this.enabled = !this.enabled;
    }
}
```
