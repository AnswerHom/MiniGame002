## 3. 界面设计

### 3.1 打开方式
- 点击主界面背包按钮打开
- 全屏弹窗
- 点击空白处关闭

### 3.2 布局结构
```
┌─────────────────────────────┐
│  背包                    ✕  │  ← 标题栏
├─────────────────────────────┤
│  [标签1] [标签2] [标签3]   │  ← 分类标签
├─────────────────────────────┤
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐     │
│  │  │ │  │ │  │ │  │     │  ← 物品网格
│  └──┘ └──┘ └──┘ └──┘     │     (4列)
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐     │
│  │  │ │  │ │  │ │  │     │
│  └──┘ └──┘ └──┘ └──┘     │
├─────────────────────────────┤
│  物品详情                  │  ← 底部详情
│  名称：xxx  数量：x        │
└─────────────────────────────┘
```

### 3.3 物品分类
| 标签 | 内容 |
|------|------|
| 全部 | 所有物品 |
| 装备 | 武器、防具、饰品 |
| 消耗 | 回血道具、BUFF道具 |
| 材料 | 进阶材料等 |

### 3.4 物品格子
- 尺寸：60×60px
- 间距：8px
- 背景：半透明深色
- 选中态：边框高亮

### 3.5 视觉风格
- 色调：深蓝紫色（仙侠风）
- 背景：半透明遮罩
- 边框：金色描边
- 图标：像素风格

---

## 4. 交互设计

### 4.1 物品操作
- 点击：选中物品
- 长按/双击：使用/装备
- 拖拽：整理背包

### 4.2 关闭方式
- 点击右上角×按钮
- 点击空白处
- 按ESC键
