# MiniGame002 v1.7.0 设计文档

## 版本主题
装备系统 - 玩家可穿戴装备提升属性

---

## 1. 需求来源

**新优化方向列表第1条**：装备系统 - 玩家可穿戴装备提升战力

---

## 2. 功能需求

### 2.1 装备类型

- **武器**：提升攻击力
- **防具**：提升防御力（可选，后续实现）
- **饰品**：提升暴击率或暴击伤害（可选，后续实现）

### 2.2 装备获取

- 怪物掉落装备
- 首次击败特定怪物掉落对应装备
- 装备掉落概率和品质随机

### 2.3 装备穿戴

- 背包界面增加装备穿戴入口
- 选中装备后点击"穿戴"按钮
- 装备属性实时生效
- 可替换已有装备

### 2.4 装备显示

- 装备图标显示
- 装备属性数值显示
- 穿戴状态标识

---

## 3. 技术实现

### 3.1 数据结构

```javascript
// 装备数据结构
{
  id: "equip_001",
  name: "铁剑",
  type: "weapon",
  icon: "weapon_sword.png",
  rarity: "common", // common, rare, epic, legendary
  attrs: { atk: 10 },
  desc: "一把普通的铁剑"
}

// 玩家装备数据
{
  weapon: null, // 当前穿戴的武器
  armor: null,
  accessory: null
}
```

### 3.2 文件结构

```
js/
  ├── equipment.js     # 装备数据和逻辑
  └── ui/backpack.js   # 集成装备穿戴功能
```
