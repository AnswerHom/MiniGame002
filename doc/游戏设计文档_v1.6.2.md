# MiniGame002 v1.6.2 设计文档

## 版本主题
小版本优化 - UI优化 + 怪物形象丰富化

---

## 1. UI界面优化

### 1.1 解决UI重叠问题
- 重新规划UI布局层级
- 装备信息、怒气条、技能栏等避免重叠
- 固定UI区域，避免动态弹窗遮挡

### 1.2 布局规范
| 区域 | 内容 |
|------|------|
| 左上 | 角色头像+等级+境界 |
| 右上 | 宗门/仙侣信息 |
| 中上 | 攻击力/暴击率显示 |
| 底部中央 | 技能栏（不与装备栏重叠） |
| 底部两侧 | 装备/背包入口 |

## 2. 怪物形象丰富化

### 2.1 现有怪物
- 阴魂、妖狼、毒蛛、僵尸

### 2.2 新增怪物（每种3种形态）
| 怪物类型 | 形态1 | 形态2 | 形态3 |
|----------|-------|-------|-------|
| 鬼魂类 | 游魂 | 恶灵 | 厉鬼 |
| 兽类 | 苍狼 | 猛虎 | 獠牙猪 |
| 虫类 | 毒蛛 | 蜈蚣 | 蝎子 |
| 尸类 | 僵尸 | 跳尸 | 铜甲尸 |

### 2.3 怪物属性差异
- 不同形态有不同颜色/体型
- 境界越高，怪物形象越高级
- 保留原有4种基础怪物
