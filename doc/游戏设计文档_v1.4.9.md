## 1. 本次更新内容

### 1.1 统一距离计量规则
- **规则定义**：游戏中 **10像素 = 1米**
- **适用范围**：UI显示的距离值、攻击范围、安全距离等所有距离相关数值
- **实现方式**：新增距离换算常量 `PIXELS_PER_METER = 10`

### 1.2 UI距离显示适配
- **距离显示**：将原有的像素距离转换为米显示
- **换算公式**：显示距离(米) = 实际像素 / 10
- **显示位置**：UI界面中的距离显示区域

### 1.3 攻击范围适配
- **玩家攻击范围**：80像素 → 8米
- **怪物攻击距离**：保持原有逻辑，确保战斗判定正确
- **安全距离**：根据新规则调整

### 1.4 配置常量新增
```javascript
// distance.js - 新增距离常量模块
const PIXELS_PER_METER = 10; // 10像素 = 1米

// 距离转换函数
function pixelsToMeters(pixels) {
    return Math.round(pixels / PIXELS_PER_METER);
}

function metersToPixels(meters) {
    return meters * PIXELS_PER_METER;
}
```
