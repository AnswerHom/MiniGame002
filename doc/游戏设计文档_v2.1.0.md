## 需求概述

灵气修炼系统是境界系统的配套玩法，玩家通过击杀怪物获得灵气，灵气用于境界突破。该系统与现有的境界系统深度整合，提供更丰富的成长体验。

---

## 功能清单

### 1. 灵气获取
- 击杀怪物获得灵气值
- 灵气值独立于经验值
- 不同境界的怪物掉落不同数量的灵气
- 灵气自动累计，无需手动操作

### 2. 灵气存储与显示
- UI显示当前灵气值和突破所需灵气
- 灵气条可视化显示进度
- 境界突破时消耗灵气

### 3. 境界突破机制
- 灵气达到突破阈值时自动提示
- 突破消耗一定数量的灵气
- 突破成功提升境界和属性
- 突破失败不消耗灵气（可选：降低成功率）

### 4. 境界与灵气对应关系
| 境界 | 突破所需灵气 | 灵气来源倍率 |
|------|-------------|-------------|
| 炼气期 | 0 (初始) | 1.0x |
| 筑基期 | 1000 | 1.2x |
| 金丹期 | 3000 | 1.5x |
| 元婴期 | 8000 | 2.0x |
| 化神期 | 20000 | 2.5x |
| 渡劫期 | 50000 | 3.0x |
| 仙人 | - | - |

### 5. UI显示
- 境界图标旁显示灵气条
- 突破提示弹窗（灵气满时）
- 灵气获取飘字效果

## 数值设计

### 灵气掉落
- 普通怪物: 10~20 灵气
- 精英怪物: 30~50 灵气
- BOSS怪物: 100~200 灵气

### 境界属性加成
| 境界 | 生命加成 | 攻击加成 | 速度加成 |
|------|---------|---------|---------|
| 炼气期 | 100% | 100% | 100% |
| 筑基期 | 120% | 115% | 105% |
| 金丹期 | 150% | 135% | 110% |
| 元婴期 | 200% | 160% | 115% |
| 化神期 | 280% | 200% | 120% |
| 渡劫期 | 400% | 280% | 130% |
