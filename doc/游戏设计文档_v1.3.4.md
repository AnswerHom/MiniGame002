## 3. 待改进需求 (v1.3.4)

### 3.1 优先级：高
1. **README自测说明**
   - 在README.md中增加游戏自测步骤
   - 说明如何启动游戏
   - 说明验收检查点

### 3.2 优先级：高
2. **去掉方向键（挂机游戏）**
   - 当前版本支持方向键控制移动
   - 问题：作为挂机游戏，不需要玩家手动控制移动
   - 解决方案：移除方向键控制，主角自动向右移动

### 3.3 优先级：中
3. **音效开关UI优化**
   - 当前已有音效开关，但位置不明显
   - 建议：添加更明显的音效图标按钮

4. **移动端触屏支持**
   - 当前仅支持键盘操作
   - 建议：添加虚拟方向键或触屏移动支持

### 3.4 优先级：低
5. **游戏帮助说明**
   - 添加操作说明UI
   - 说明攻击方式、升级机制
