# MiniGame002 v1.5.0 设计文档

## 版本主题
战斗体验优化 + UI界面重构

---

## 1. 战斗体验优化

### 1.1 战斗状态机
- **推进状态**：主角自动前进
- **遇怪停止**：检测到怪物时自动停下
- **进入战斗**：切换为战斗状态
- **战斗胜利**：继续推进

### 1.2 战斗场景分离
- 战斗场景：固定视角，怪物从右侧涌入
- 推进场景：横版卷轴，主角自动行走
- 场景切换动画过渡

### 1.3 战斗节奏优化
- 攻击间隔：1.2秒/次
- 怪物刷新：每波3只，间隔5秒
- 战斗时长：单局2-3分钟

---

## 2. UI界面优化

### 2.1 界面整合
| 区域 | 内容 |
|------|------|
| 左上角 | 角色信息（头像、等级、境界） |
| 右上角 | 宗门/仙侣信息 |
| 底部中央 | 技能栏（整合所有技能） |
| 底部两侧 | 装备/背包入口 |

### 2.2 视觉风格
- 统一UI色调（深色系）
- 图标风格一致
- 减少文字堆砌，多用图标表达
- 重要信息高亮，次要信息弱化

### 2.3 交互优化
- 点击区域扩大
- 反馈动画增强
- 减少弹窗使用
