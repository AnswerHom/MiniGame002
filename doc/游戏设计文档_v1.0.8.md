# MiniGame002 v1.0.8 设计文档

## 版本主题
放置挂机优化 + 主角形象优化

---

## 1. 游戏类型确认

### 1.1 核心玩法
- **放置挂机游戏**
- 主角自动战斗，无需玩家操作
- 自动前进、自动攻击、自动升级

### 1.2 控制方式
- **无需键盘控制**
- 无需手动移动
- 无需手动攻击
- 纯自动化挂机体验

---

## 2. 主角形象优化

### 2.1 基础造型
- **整体风格**：简约水墨风修仙角色
- **体型**：Q版可爱风格，头身比1:1.5
- **高度**：48px（略大一些，存在感更强）

### 2.2 配色方案
| 部位 | 颜色 | 色值 |
|------|------|------|
| 衣服 | 月白色长袍 | #f0f5f9 |
| 衣领袖口 | 淡青色 | #81e6d9 |
| 腰带 | 深青色 | #319795 |
| 头发 | 墨黑色 | #1a202c |
| 发带 | 红色细绳 | #fc8181 |
| 皮肤 | 暖白色 | #fffaf0 |
| 鞋子 | 灰色 | #a0aec0 |

### 2.3 武器外观
- **剑**：银色剑身(#cbd5e0)，剑柄深灰(#4a5568)，剑穗红色
- **刀**：黑色刀身(#2d3748)，金色刀柄(#d69e2e)
- **枪**：金色枪头(#ffd700)，木色枪杆(#b7791f)

### 2.4 头部细节
- 眼睛：大而有神，黑白分明
- 表情：略带微笑
- 头饰：简单的发髻+红色发带飘带

---

## 3. 动画效果

### 3.1 待机动画
- 轻微上下浮动，幅度2px，周期2秒
- 衣服衣摆微微飘动

### 3.2 战斗攻击动画（自动）
- 攻击时武器自动挥动
- 挥剑：向右挥动150度
- 武器挥动轨迹有淡白色残影
