# MiniGame002 v3.7.0 设计文档

## 版本主题
右侧系统入口布局优化

---

## 1. 需求来源

根据未实现需求列表：右侧系统入口布局优化，不要超出屏幕外

---

## 2. 问题分析

### 2.1 当前问题
- 右侧系统入口按钮超出屏幕
- 按钮被截断
- 影响用户体验

### 2.2 期望效果
- 优化布局，不超出屏幕
- 整齐排列
- 保持功能完整

---

## 3. 优化方案

### 3.1 布局调整
- 按钮整体左移，进入安全区域
- 确保在屏幕内完整显示
- 保持按钮可点击

### 3.2 尺寸调整
- 按钮视觉尺寸：36×36px（参考v3.3.0）
- 按钮间距：10-15px
- 整体高度控制在安全范围内

### 3.3 位置坐标
- X坐标：屏幕宽度 - 按钮宽度 - 安全边距(20px)
- Y坐标：依次排列，从上往下

---

## 4. 验收标准

### ✅ 功能验收
- [x] 按钮完整显示，不超出屏幕
- [x] 点击功能正常
- [x] 布局整齐
