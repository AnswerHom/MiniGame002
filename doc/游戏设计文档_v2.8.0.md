# MiniGame002 v2.8.0 设计文档

## 版本主题
主界面精简 - 删除无用信息

---

## 1. 需求来源

根据未实现需求列表第3条：主界面精简，删除无用信息，只保留主角等级、经验条、金币、距离，不显示波次、累计次数、伤害值

---

## 2. 问题分析

### 2.1 当前问题
- 主界面显示信息过多
- 存在无用的信息显示
- UI显得杂乱

### 2.2 期望效果
- 精简主界面
- 只保留必要信息
- UI清晰整洁

---

## 3. 精简设计

### 3.1 保留信息
| 信息 | 位置 |
|------|------|
| 主角等级 | 左上角 |
| 经验条 | 等级下方 |
| 金币 | 右上角 |
| 距离 | 右上角（距离下方） |

### 3.2 删除信息
- 波次显示
- 累计次数
- 伤害值统计

### 3.3 界面布局
```
左上角          右上角
[等级]          [金币]
[经验条]        [距离]

            中央（游戏区域）
```

---

## 4. 验收标准

### ✅ 精简验收
- [x] 只保留：等级、经验条、金币、距离
- [x] 删除：波次、累计次数、伤害值
- [x] UI布局清晰整洁
