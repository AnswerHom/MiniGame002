## 2. v1.0.1 更新内容

### 2.1 新增功能

| 功能 | 说明 |
|-----|-----|
| **境界系统** | 玩家每升5级突破一个境界（筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫） |
| **闪避系统** | 受伤后0.5秒无敌闪避时间 |
| **云雾效果** | 背景增加动态云雾，增强修仙氛围 |
| **星空背景** | 夜晚星空效果 |
| **距离显示** | 显示玩家前行距离 |

### 2.2 UI优化

- 右上角显示当前境界
- 显示前行距离（米）
- 血量归零时显示为0而非负数
- 闪避状态提示

### 2.3 视觉优化

- 深紫色修仙风格天空
- 动态云雾漂浮
- 闪烁星星背景
- 灵草装饰地面
- 仙剑宗风格山脉
