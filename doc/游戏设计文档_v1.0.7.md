# MiniGame002 v1.0.7 设计文档

## 版本主题
主角形象优化

---

## 1. 主角外观设计

### 1.1 基础造型
- **整体风格**：简约水墨风修仙角色
- **体型**：Q版可爱风格，头身比1:1.5
- **高度**：48px（略大一些，存在感更强）

### 1.2 配色方案
| 部位 | 颜色 | 色值 |
|------|------|------|
| 衣服 | 月白色长袍 | #f0f5f9 |
| 衣领袖口 | 淡青色 | #81e6d9 |
| 腰带 | 深青色 | #319795 |
| 头发 | 墨黑色 | #1a202c |
| 发带 | 红色细绳 | #fc8181 |
| 皮肤 | 暖白色 | #fffaf0 |
| 鞋子 | 灰色 | #a0aec0 |

### 1.3 武器外观
- **剑**：银色剑身(#cbd5e0)，剑柄深灰(#4a5568)，剑穗红色
- **刀**：黑色刀身(#2d3748)，金色刀柄(#d69e2e)
- **枪**：金色枪头(#ffd700)，木色枪杆(#b7791f)

### 1.4 头部细节
- 眼睛：大而有神，黑白分明
- 表情：略带微笑
- 头饰：简单的发髻+红色发带飘带

---

## 2. 动画效果

### 2.1 待机动画
- 轻微上下浮动，幅度2px，周期2秒
- 衣服衣摆微微飘动

### 2.2 行走动画
- 身体左右轻微摇摆
- 发带随风飘动

### 2.3 攻击动画
- 挥剑：向右挥动150度
- 武器挥动轨迹有淡白色残影

---

## 3. 主角属性（保持不变）

| 属性 | 数值 |
|------|------|
| 攻击力 | 10 |
| 攻击范围 | 80px |
| 暴击率 | 5% |
