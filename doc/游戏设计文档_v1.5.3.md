# MiniGame002 v1.5.3 设计文档

## 版本主题
美术表现优化 - 主角形象优化与怪物类型扩展

---

## 1. 优化需求来源

根据优化方向列表：美术表现（优先）
- **优化主角形象**（像素风、仙侠风）
- **完善主角攻击动作**
- **增加怪物类型**
- **优化怪物形象**

---

## 2. 问题分析

### 2.1 当前问题
- 主角形象较为简单，缺乏特色
- 怪物种类单一，只有基础几种类型
- 攻击动作不够丰富
- 与仙侠题材契合度不够

### 2.2 影响
- 玩家视觉体验单调
- 缺乏收集和探索乐趣
- 战斗缺乏多样性

---

## 3. 解决方案

### 3.1 主角形象优化

**优化方向**：
- 采用更精致的像素风格
- 增加服饰细节（衣纹、飘带）
- 优化发型设计，增加仙侠感
- 保留水墨画风格

**实现要点**：
```javascript
// player.js - 优化主角绘制
const PLAYER_APPEARANCE = {
    robeDetail: true,      // 增加衣服细节
    ribbon: true,          // 增加飘带效果
    hairDetail: true,      // 精细头发
    style: 'xianxia'      // 仙侠风格
};
```

### 3.2 主角攻击动作优化

**攻击动作丰富**：
- 添加攻击前摇动作（蓄力）
- 添加攻击后摇动作（收招）
- 不同武器类型对应不同攻击动作
- 攻击时身体配合动作

**动作参数**：
```javascript
const ATTACK_ANIMATION = {
    windUp: 0.15,      // 前摇时间(秒)
    strike: 0.1,        // 攻击时间(秒)
    followThrough: 0.1, // 后摇时间(秒)
    total: 0.35         // 总时长
};
```

### 3.3 怪物类型扩展

**新增怪物类型**：

| 怪物名 | 境界 | 生命值 | 攻击力 | 经验值 | 特点 |
|--------|------|--------|--------|--------|------|
| 野猪精 | 练气期 | 80 | 15 | 30 | 皮厚移动慢 |
| 灰狼 | 练气期 | 60 | 20 | 35 | 速度快 |
| 蛇妖 | 筑基期 | 120 | 25 | 60 | 有毒性攻击 |
| 僵尸 | 筑基期 | 150 | 30 | 80 | 高血量 |
| 山贼 | 筑基期 | 100 | 35 | 70 | 攻击高 |

**怪物选择逻辑**：
- 根据玩家境界和距离动态生成
- 境界越高，怪物越强
- 随机选择同境界不同类型

```javascript
const ENEMY_POOL = {
    '炼气期': ['野猪精', '灰狼', '兔子'],
    '筑基期': ['蛇妖', '僵尸', '山贼'],
    '金丹期': ['虎精', '狐妖', '骷髅']
};
```

### 3.4 怪物形象优化

**每种怪物独特外观**：
- 不同的体型和配色
- 独特的攻击动作
- 死亡时有倒地动画

```javascript
const ENEMY_APPEARANCE = {
    '野猪精': { body: 'oval', color: '#8B4513', tusks: true },
    '灰狼': { body: 'wolf', color: '#808080', ears: true },
    '蛇妖': { body: 'serpent', color: '#228B22', tongue: true }
};
```

## 4. 技术实现

### 4.1 主角绘制优化

```javascript
// player.js - draw() 方法优化
draw() {
    // 绘制优化后的身体
    this.drawBody();
    
    // 绘制衣服细节
    this.drawRobeDetail();
    
    // 绘制飘带
    this.drawRibbon();
    
    // 绘制武器（带攻击动画）
    this.drawWeapon();
    
    // 绘制头发
    this.drawHair();
}
```

### 4.2 攻击动作状态机

```javascript
// player.js - 攻击动作状态
const ATTACK_STATES = {
    IDLE: 'idle',
    WIND_UP: 'windUp',      // 蓄力前摇
    STRIKE: 'strike',       // 攻击
    FOLLOW_THROUGH: 'followThrough', // 后摇
    COOLDOWN: 'cooldown'    // 冷却
};
```

### 4.3 怪物生成扩展

```javascript
// enemy.js - 扩展怪物生成
function spawnEnemy(playerRealm, distance) {
    const pool = ENEMY_POOL[playerRealm] || ENEMY_POOL['炼气期'];
    const type = pool[Math.floor(Math.random() * pool.length)];
    return new Enemy(type, distance);
}
```
