## 2. 远景元素（待实现）

### 2.1 天空
- **白天**：浅蓝色渐变（#87ceeb → #e0f7fa）
- **黄昏**：橙粉色渐变（#ff7e5f → #feb47b）
- **夜晚**：深蓝色（#0c1445）配星星

### 2.2 云朵
- 白色半透明云朵（rgba(255,255,255,0.6)）
- 随机大小：80-150px
- 缓慢飘动动画
- 每5-10秒生成新云朵

### 2.3 星星（夜晚）
- 白色小亮点，大小2-4px
- 随机分布在天空中
- 闪烁效果（透明度变化）
- 数量：20-30颗
